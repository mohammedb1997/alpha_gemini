# 🏆 تحدي الحروف - لعبة المسابقات الذكية

لعبة تفاعلية بنظام الخلايا السداسية، تجمع بين المعرفة وسرعة البديهة. تدعم نمطين للعب: **محلي** (جهاز واحد) و **مشترك** (متعدد الشاشات).

## 📌 محتويات المشروع

- `index.html` - الصفحة الرئيسية لاختيار نمط اللعب
- `normal.html` - نمط اللعب المحلي (لاعبان على نفس الجهاز)
- `judge.html` - شاشة الحكم في النمط المشترك
- `display.html` - شاشة العرض (البروجكتر) في النمط المشترك
- `player.html` - شاشة البازر للاعبين (على الهواتف)
- `style.css` - التنسيقات المشتركة
- `qbank.js` - بنك الأسئلة (40 سؤال لكل حرف)
- `howto.html` - دليل الاستخدام التفصيلي
- `README.md` - هذا الملف

## 🎮 كيفية التشغيل

### 1️⃣ النمط المحلي (جهاز واحد)
1. افتح ملف `index.html` واختر **"اللعب المحلي"**.
2. ستظهر لوحة الخلايا السداسية مع لوحة النتائج.
3. اضغط على أي حرف لفتح نافذة السؤال.
4. بعد قراءة السؤال:
   - استخدم **العداد التنازلي 20 ثانية**.
   - اضغط **"إظهار الإجابة"** لمعرفة الحل.
   - اضغط **"تغيير السؤال"** لسؤال آخر بنفس الحرف.
   - حدد الفائز بالضغط على **"فوز الأزرق"** أو **"فوز الأحمر"**.
5. يفوز الفريق الذي يربط بين جانبي اللوحة أولاً (أعلى-أسفل للأزرق، يسار-يمين للأحمر).

### 2️⃣ النمط المشترك (متعدد الشاشات)
1. افتح `index.html` واختر **"اللعب المشترك"** لفتح شاشة الحكم (`judge.html`).
2. في شاشة الحكم:
   - أدخل أسماء الفريقين.
   - اضغط **"إنشاء الغرفة وبدء اللعبة"**.
   - سيظهر لك **رابط شاشة البروجكتر** و **QR كود للاعبين**.
3. افتح الرابط الأول في نافذة جديدة (شاشة العرض/البروجكتر).
4. اطلب من اللاعبين مسح QR كود بهواتفهم، واختيار الفريق (أ أو ب).
5. الآن:
   - **الحكم** يختار خلية من اللوحة ليظهر السؤال.
   - **اللاعبون** يضغطون البازر في هواتفهم للإجابة.
   - **شاشة البروجكتر** تعرض السؤال وحالة البازر.
   - الحكم يقيّم الإجابة ويمنح النقطة للفريق الصحيح.

## ❓ الأسئلة الشائعة

### س: اللعبة لا تعمل على GitHub Pages؟
**ج:** تأكد من رفع **جميع الملفات** في نفس المجلد، وأن اسم الملف الرئيسي هو `index.html`.

### س: لماذا لا يتصل اللاعبون في النمط المشترك؟
**ج:** قد تكون شبكة أحد الأطراف مقيدة. الحل:
- تأكد من استخدام **نفس شبكة Wi-Fi** للتجربة الأولية.
- تم تضمين **TURN Server** مجاني ليعمل عبر الإنترنت، لكن قد يكون بطيئاً. يفضل استخدام شبكة محلية.

### س: كيف أضيف أسئلة جديدة؟
**ج:** افتح ملف `qbank.js` وأضف أسئلة بصيغة:
```javascript
"الحرف": [
  { q: "السؤال؟", a: "الإجابة" },
  // ...
]
